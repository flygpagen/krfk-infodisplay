# InfoDisplay flygklubb
Ett projekt byggt med Lovable. 

## Beskrivning
Detta är en enkel informationspanel för en flygklubb som hämtar bokningar från myweblog och metar/taf från checkwx.com. Båda kräver att man har API-tillgång. Nycklar placeras i config.php. ESMK är hårdkodat men går säkert att enkelt ändra.

## What technologies are used for this project?
This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## How can I deploy this project?

Hämta med git clone, bygg med npm run build och flytta sen innehållet i /dist till din webbkatalog.
